## The enviroment
```
sudo apt install ruby-dev
sudo gem install jekyll bundler
```
## Run and browse the website
```
bundle exec jekyll serve
```
- [http://localhost:4000](http://localhost:4000)
## Publish
```
git push -u origin main
```
